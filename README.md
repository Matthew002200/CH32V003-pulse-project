# Генерация импульсов на CH32V003
Учебный проект генерации импульсов для силовой электроники с использованием микроконтроллера CH32V003 и SOS-эффекта (semiconductor opening switch).

## Цель проекта
Изучение основ программирования микроконтроллеров и формирования импульсов заданной длительности.

## Описание проекта
В проекте используется микроконтроллер CH32V003 для управления формированием импульсов, предназначенных для управления полупроводниковым размыкателем (SOS), реализованным на диоде в импульсном режиме.
Проект выполнялся самостоятельно в рамках изучения embedded-разработки.

## Текущее состояние
- Разработана электрическая схема  
- Написан прототип прошивки на языке C  
- Возникли сложности с точной установкой времени импульсов
